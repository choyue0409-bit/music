# 现场手账 · Live Memory

一个记录你看过的演唱会和音乐节的小工具。
按歌手 / 时间 / 地图三种方式回顾你的现场记忆，存照片、视频、票根和当天的小作文。

## 功能

- 添加 / 编辑 / 删除一场演出（歌手、日期、城市、场馆、票价、同行的人、评分）
- 上传任意张数的照片和视频（存在你的浏览器里）
- 把某张照片标记为「票根」，单独归类
- 写当天的回忆 / 小作文
- 三个视图：
  - **歌手**：按歌手聚合，看你最常去的人
  - **时间轴**：按年份分组
  - **地图**：在世界地图上看你去过的现场（需要填写经纬度）
- 导出 / 导入备份（JSON 文件，含照片和视频）

## 怎么跑起来

需要 Node.js 18 或更高版本。第一次使用：

```bash
npm install
npm run dev
```

然后浏览器打开终端里给出的网址（通常是 http://localhost:5173 ）。

## 怎么打包发布

```bash
npm run build
```

会生成一个 `dist/` 文件夹。可以把它丢到任何静态网站托管（Vercel、Netlify、Cloudflare Pages 都行，且免费）。

## 数据存哪里

- 文字信息（歌手名、日期、回忆等）：浏览器的 `localStorage`
- 照片和视频：浏览器的 `IndexedDB`
- **不会上传到任何服务器**。换浏览器或清除浏览器数据会丢失，记得在「设置」里定期导出备份。
